## Description
A simple REST API.

## Project setup

```bash
$ yarn
```

## Compile and run the project

```bash
# development mode
$ yarn run start:dev

# production mode
$ yarn build && yarn start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Decisions

- Fastify is used instead of express because who does not like faster response times?
